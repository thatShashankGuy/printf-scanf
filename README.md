**Quick Start:**
```bash

cd blog-cli && bun install

# Configure your .env file (LLM API required for AI features)
cp blog-cli/.env.example .env
# Edit .env with your API credentials

# List existing posts
bun run cli list

# Create a new post with AI
bun run cli new --ai

# Preview locally
bun run cli preview

# Publish to GitHub
bun run cli publish
```
